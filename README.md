# Kritajna – Local Commerce Platform

A **Django-based local commerce platform** that connects **farmers (sellers)**, **customers**, and **delivery partners** in a transparent, user-friendly ecosystem.

---

## 🌟 Overview
Kritajna empowers farmers by giving them direct access to customers and delivery partners.  
It ensures **fair trade, transparency, and ease of use** for all stakeholders.

---

## 👨‍🌾 Features by Role

### Farmers
- Kissan-ID based authentication
- One-time subscription model
- Product management (add, edit, delete)
- Order management & tracking
- Dashboard: sales stats, earnings, pending orders
- Payment history (subscriptions + earnings)

### Customers
- Browse products by name or location
- Detailed product info with farmer details
- Easy ordering with cart + delivery address
- Order tracking (pending → delivered)
- Order history & spending overview

### Delivery Partners
- Assigned orders for delivery
- Pickup & delivery location tracking
- Status updates (picked up → delivered)
- Earnings dashboard

---

## ⚙️ Tech Stack
- **Backend**: Django 5.2  
- **Database**: SQLite (upgradeable)  
- **Frontend**: HTML5, CSS3, Bootstrap 5  
- **Image Handling**: Pillow  
- **Server**: Django Development Server
